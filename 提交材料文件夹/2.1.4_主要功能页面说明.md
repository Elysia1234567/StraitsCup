## 2.1.4 主要功能页面说明

本项目主要功能页面包括首页 Home、Agent 对话页、智能体库页、Models 三维展示页和 Graph 关系图谱页，分别对应“入口浏览、智能对话、智能体选择、形象展示、关联探索”五类核心体验。各页面围绕统一的深色科技风视觉体系进行设计，既保证风格一致，也便于用户在不同功能之间快速切换。

首页 Home 作为作品入口，主要用于展示福建非遗总览和项目主视觉。页面采用地图为中心的布局方式，用户可以通过点击地图中的区域或下方卡片进入对应非遗内容，也可以通过顶部导航切换到其他功能页。该页面的设计特色是首屏聚焦强、文化识别度高，操作方式以点击和悬停浏览为主，适合快速建立整体印象。

Agent 对话页是系统的核心交互页面，主要用于多智能体群聊、知识问答和证据链查看。页面采用左侧成员列表、中间对话区、右侧知识面板的三栏布局，用户可以添加或切换 Agent，围绕非遗主题进行持续交流，并查看回答依据与相关信息。该页面强调实时交互和流式回复，操作方式以输入提问、选择成员、查看知识结果为主。

智能体库页用于集中浏览和选择可加入聊天室的智能体，展示各个 Agent 的头像、人设标签、简介和适用主题，帮助用户快速找到合适的对话成员。页面设计强调卡片化呈现和主题分类，操作方式以浏览、筛选和点击添加为主，既方便首次使用者快速上手，也便于在不同非遗主题下灵活组建对话阵容。

Models 页面用于展示非遗相关的三维模型和形象内容，帮助用户从视觉上理解具体工艺和文化对象。页面以沉浸式展示为主，支持模型浏览、旋转和观察，整体风格偏向简洁展示，突出模型本体。用户进入页面后，可通过鼠标拖拽或视角切换查看不同角度，增强对非遗对象的直观感受。

Graph 页面用于展示福建非遗项目之间的关联关系，帮助用户从整体上理解不同地区、不同项目之间的联系。页面采用关系图谱形式组织内容，节点分布清晰，视觉上具有较强的层次感和网络感。用户可通过点击节点查看对应信息，通过缩放和拖拽浏览图谱结构，适合进行文化脉络探索和主题延展。

手机端图鉴页用于浏览非遗图像与代表性内容，页面以卡片浏览和竖屏滑动为主，将不同非遗项目的图片、名称和简要信息集中展示，便于用户在手机上快速查看和切换内容。该页延续统一的深色科技风视觉，并通过较大的图片区域和简洁的文字信息降低阅读负担，适合移动场景下进行快速浏览。

手机端智能体页用于集中浏览和选择可加入对话的智能体，页面以卡片网格形式展示头像、人设标签和简要介绍，帮助用户在手机上快速筛选合适的成员。该页强调轻量化浏览和快速添加，用户可通过滑动查看不同智能体，并点击卡片完成选择操作，方便在移动端灵活组建对话阵容。

手机端 Agent 对话页是移动端的主要交互页面，采用上下分区的聊天布局，将对话内容和输入区域集中在同一屏幕内，便于用户在手机上完成提问、查看回复和继续追问。页面重点突出消息流和输入操作，同时保留必要的功能按钮，使多智能体对话在小屏幕上仍能保持清晰、顺畅的使用体验。

总体来看，这五类页面共同构成了作品的主功能链路：首页负责引入，Agent 页负责对话，智能体库页负责选人，Models 页负责展示，Graph 页负责关联。页面之间通过顶部导航相互连接，形成统一、流畅、易操作的使用体验。
