## 2.4 作品创新点

创新点一：**非遗对象角色化**。项目将寿山石雕、脱胎漆器、提线木偶、德化瓷、漳州木版年画等非遗项目封装为具备人设、语气、知识边界与生成约束的文化智能体（Agent），让非遗从静态条目变为可被召唤、可被追问的"数字生命"，形成稳定的文化记忆点。

创新点二：**多智能体圆桌协作**。系统支持在同一聊天室内组合"历史学家、老匠人、年轻游客"等多种角色围绕同一主题分工互补，并通过主题匹配与发言节奏控制避免抢答与重复输出，使非遗讲解从"一问一答"升级为"圆桌讨论"。

创新点三：**用户介入式群聊**。用户可在多智能体讨论中随时插话或追问，系统会根据问题语义自动选择最合适的 Agent 进行回应，让用户从旁观者转变为讨论的参与者。

创新点四：**证据链与置信度展示**。每一条对话都同步生成知识摘要、关键证据、综合置信度与溯源信息，并在右侧面板与回答同屏呈现，使非遗问答从"生成式表达"走向"可解释表达"。

创新点五：**RAG 双轨与本地回退**。知识检索同时支持本地知识库与 Milvus 向量增强，当外部向量环境不可用时自动回退到本地相似度检索，保障答辩现场不因单点依赖而中断。

创新点六：**AIGC 生图回流聊天室**。生成的图像不再是一次性输出，而是以消息形式回流到当前聊天室，成为后续讨论的新对象，形成"理解 → 创作 → 反馈 → 再创作"的对话内共创循环。

创新点七：**非遗展示形象化**。系统在问答过程中会根据当前讨论的非遗对象，自动调取匹配的高清图片与基于 OGL（WebGL）渲染的 3D 模型（如寿山石雕摆件、提线木偶造型、德化瓷瓶等），与文字回答同屏呈现并支持旋转、缩放观察，让用户在听讲工艺、纹样与形制的同时直接"看见"实物，使非遗讲解从"听说"升级为"亲见"。

创新点八：**面向高并发的工程稳定性**。基于 Redis 消息队列、异步任务调度与 WebSocket 流式推送，把耗时的大模型对话与图像生成统一异步化，支撑多评委同时操作下的稳定演示与逐字流式响应。

创新点九：**多端一致的文化叙事**。Web 端、UniApp 移动端与 Three.js / OGL 沉浸式页面共享同一套内容逻辑与视觉语言，让答辩大屏、手持移动和展厅 3D 展示在同一文化叙事下自然衔接。

## 2.5 作品实用价值

实用价值一：**教育场景**。可作为高校非遗通识课、中小学研学与文化实践的互动工具——教师以非遗主题创建聊天室，学生与多位文化智能体共同讨论，并通过证据链查看回答依据，把"被动听讲"转变为"带着证据追问"。

实用价值二：**展览场景**。可作为博物馆、非遗馆与文化馆的智能讲解中枢——观众在展品前向对应智能体提问，由"数字传承人"以闽南语、福州话等方言进行讲解，并通过 AIGC 生图与 3D 展厅获得共创式参观体验。

实用价值三：**文旅传播**。以福建地图为入口，天然契合地方文旅推介需求，游客可在出行前浏览各地市非遗资源，并通过关系图谱了解跨地市的同源非遗项目；更进一步，系统为各地非遗精心设计了具有地域辨识度的虚拟形象与人设（如莆田妈祖、泉州木偶、德化瓷娘等），借助二次元化、IP 化的视觉表达吸引年轻用户主动"为角色奔赴一座城"，把线上对话兴趣转化为线下打卡、研学与参观的真实文旅流量，为地方非遗带来年轻一代的关注与传播。

实用价值四：**创意设计**。可辅助设计师与内容创作者完成海报、文创、角色设定与活动方案的前期构思，借助风格化生图快速生成符合非遗审美原型的视觉草图，作为后期手绘或精修的起点。

实用价值五：**非遗传承**。面对部分非遗项目"老匠人难寻、年轻人难入门"的困境，系统可作为非遗技艺的数字化抢救与活态传承工具——通过结构化采集老匠人讲解的工艺流程、口诀心法与代表作品，沉淀为可对话的"数字传承人 Agent"，让濒危技艺的知识不再依赖个人记忆而能持续被追问、复现与传习；同时借助方言讲解、3D 实物展示与 IP 化形象，把原本枯燥的工艺细节转化为年轻一代愿意主动了解、模仿与再创作的内容，为非遗传承注入新的代际入口。

实用价值六：**长期平台化潜力**。系统已形成"地图入口 → 成员管理 → 多智能体对话 → 知识洞察 → 图像生成 → 归档展示"的完整闭环，内容可扩展、能力可插拔、部署可伸缩、数据可沉淀，具备由比赛作品演化为长期非遗数字传播平台的基础。
