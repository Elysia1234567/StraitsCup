# 同源 OmniSource 界面与交互场景提示词

## 统一风格基准

所有画面保持一致：深蓝数字科技风、半透明玻璃面板、蓝紫发光边框、点阵纹理、细线数据流，结合福建非遗材质色彩。界面应像可真实落地的 Vue 3 + Vite 产品原型展示，专业、克制、清晰，避免营销海报感和过多说明文字。

## 1. Web 端 Agent 对话页三栏布局

**用途**：核心功能页面展示。

**提示词**：
一张 Web 端非遗多智能体聊天界面概念图，16:9 桌面屏比例，左侧是聊天室列表和 Agent 成员列表，中间是流式对话消息区，右侧是知识增强与证据链面板；界面主题为深蓝科技风，发光边框、半透明玻璃卡片、紫蓝数据流；Agent 头像体现福建非遗，如寿山石雕、德化瓷、提线木偶、木版年画；消息流中有文本消息、图片生成结果卡片、加载中的流式回复光标；右侧面板展示抽象证据节点和置信度图形，不要生成可读文字，专业产品 UI，高清。

**反向提示词**：
聊天软件截图复刻，杂乱小字，白底办公软件，低端仪表盘，过度圆角卡片堆叠。

## 2. Web 端 Agent 库页

**用途**：展示智能体浏览、筛选、添加功能。

**提示词**：
深蓝科技风 Web 端 Agent 库页面，网格卡片展示多个福建非遗文化智能体，每张卡片包含头像、非遗材质装饰、标签区域和添加按钮位置但不要可读小字；角色包括福州寿山石雕器灵、泉州提线木偶器灵、德化瓷器灵、漳州木版年画器灵、厦门珠绣器灵、宁德剪纸器灵；顶部有搜索筛选栏，卡片发光描边，背景点阵纹理与传统纹样暗纹融合，整体像高质量产品展示截图，16:9。

**反向提示词**：
人物风格不统一，卡片拥挤，儿童卡牌游戏风，纯白背景，文字乱码。

## 3. Web 端 Graph 关系图谱页

**用途**：展示非遗关系图谱与文化脉络探索。

**提示词**：
一个深蓝色 Web 端关系图谱界面，中心节点为“福建非遗数字生命平台”的抽象光核，周围连接九地市节点和非遗项目节点，节点用小图标或材质球表示：瓷白、朱红年画、玉石绿、金色漆器、木偶细线、剪纸红；用户点击某个节点后右侧出现信息面板，但不显示可读文字；图谱层次清晰，节点发光，线条有数据流动感，背景为点阵星图和传统云纹暗纹，16:9，高级可视化风格。

**反向提示词**：
普通社交网络图，节点过密看不清，颜色混乱，白底科研论文图，低清截图。

## 4. Web 端 3D / WebGL 虚拟展馆

**用途**：展示 Models 页、3D 展厅与沉浸展示能力。

**提示词**：
深蓝数字展馆中的 WebGL 虚拟展厅，中央陈列可旋转观察的非遗三维模型：寿山石雕摆件、德化白瓷瓶、泉州提线木偶、建盏、脱胎漆器；展品放在发光圆形展台上，周围有半透明 UI 控件、视角旋转轨迹和模型信息面板占位，无可读文字；空间有蓝紫光带、金色纹样、微弱粒子流，突出“可旋转、可缩放、可沉浸观看”，16:9，电影级渲染，高清。

**反向提示词**：
空展厅，展品模糊，纯博物馆照片，过度霓虹，奇幻魔法风，模型比例怪异。

## 5. 手机端 Agent 对话页

**用途**：移动端核心交互页面展示。

**提示词**：
一张竖屏手机界面概念图，展示同源 OmniSource 手机端 Agent 对话页，顶部是深蓝标题栏和当前聊天室状态，中间是清晰的聊天消息流，消息包含用户提问、多位非遗智能体头像回复、图片生成结果缩略图，底部是输入栏和少量图标按钮；整体延续深蓝科技风、紫蓝光效、半透明卡片、非遗纹样暗纹；布局适合单手操作，阅读清晰，界面真实可用，9:16，高清，无乱码文字。

**反向提示词**：
桌面网页塞进手机，文字太小，按钮重叠，浅色社交软件风，低清 UI。

## 6. 手机端图鉴与智能体选择页

**用途**：展示移动端图鉴浏览、智能体选择。

**提示词**：
竖屏手机应用界面，展示非遗图鉴与智能体选择的混合场景：上方是搜索和分类筛选区，中间为两列卡片网格，卡片展示非遗图片和器灵头像，如德化瓷、寿山石雕、提线木偶、木版年画、剪纸、畲族服饰；底部有简洁导航栏图标，整体为深蓝科技风、发光圆角边框、文化纹样暗纹、图片质感真实，适合移动端快速浏览，9:16，高清，避免可读小字。

**反向提示词**：
营销落地页，图片裁切严重，卡片过多拥挤，儿童插画风，底部导航重叠。
