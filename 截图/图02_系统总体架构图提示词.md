# 图02 系统总体架构图提示词

## 用途

用于“设计原理”或“技术架构与技术路线”章节，展示系统整体闭环。

## 生成提示词

请生成一张横版系统架构图，主题为“同源（OmniSource）系统总体架构”。画面应像一张高质量技术说明图，结构清晰、模块分层明确、适合论文/报告插图。

必须包含的模块：
- 用户端
- Vue 3 / UniApp 前端
- REST API
- WebSocket 实时通信
- 多智能体编排
- RAG 知识检索
- AIGC 图像生成
- 图像理解
- 语音识别
- MySQL
- Redis
- Milvus（可选增强）
- OSS

结构建议：
- 上层是用户和前端
- 中层是后端服务与 AI 编排
- 下层是数据层、向量库和对象存储
- 用箭头展示“本地知识库 -> RAG 检索 -> Agent 编排 -> 回复 -> 生图 -> 归档 -> 前端呈现”的流程

视觉要求：
- 线框式、扁平化、专业、干净
- 主色深蓝，辅以青色、紫色、少量绿色
- 中文标注清晰，字体统一
- 适合打印，不要做成炫酷游戏界面

负面约束：
- 不要花哨背景
- 不要过多渐变球
- 不要文字过密
- 不要把模块画得像商业广告图
