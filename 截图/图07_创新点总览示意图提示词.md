# 图07 创新点总览示意图提示词

## 用途

用于“创新点”章节，概括项目的核心创新。

## 生成提示词

请生成一张横版示意图，主题为“同源（OmniSource）创新点总览”。画面应像正式答辩材料中的总结图，不要像广告海报。

画面建议包含 4 到 5 个创新模块：
- 非遗文化 Agent 化
- 多智能体协同问答
- 本地 RAG + 可选 Milvus 增强
- AIGC 生图入聊
- Web / 移动端 / 3D 多端呈现

表现方式建议：
- 中央是项目名称与核心闭环
- 周围环绕 4 到 5 个模块卡片
- 每个模块配一行简洁说明
- 用箭头或环形关系表示“知识 -> 对话 -> 生成 -> 展示”的关系

视觉要求：
- 结构清晰、专业、留白充足
- 颜色延续深蓝、青绿、少量金色点缀
- 适合印在说明书中

负面约束：
- 不要做成复杂商业海报
- 不要出现大量长句
- 不要让模块之间互相遮挡
