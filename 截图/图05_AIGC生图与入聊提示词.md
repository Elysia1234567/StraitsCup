# 图05 AIGC 生图与入聊提示词

## 用途

用于展示“Agent 生图”能力，建议放在“创新点”或“测试与运行效果”章节。

## 生成提示词

请生成一张横版示意图，展示“文化 Agent 生成图像并进入聊天室”的闭环。画面应像产品功能示意图，而不是艺术海报。

画面建议包含：
- 左侧是当前 Agent 聊天室，用户选择一个房间内 Agent 并输入图像生成请求
- 中间是 Agent 人设、头像/原型图、房间上下文、提示词组合过程
- 右侧是生成完成的图片结果卡片
- 下方或旁侧是聊天室消息流，显示图片作为一条消息进入会话
- 可加入 `image_generation_task` 任务状态、Redis 进度、OSS 归档、WebSocket 广播等辅助元素

视觉要求：
- 强调“从提问到生成到入聊”的完整链路
- 强调生成结果会写入 `chat_message`，并通过 `/ws/chat` 广播到当前聊天室
- 图片结果区域要清楚，像展示卡片
- 风格统一，正式、科技、清晰

负面约束：
- 不要把生成结果做成随机插画集合
- 不要过于抽象
- 不要过多霓虹装饰
