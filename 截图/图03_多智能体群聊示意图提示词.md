# 图03 多智能体群聊示意图提示词

## 用途

用于“设计原理”或“测试与运行效果”章节，展示多智能体协作问答场景。

## 生成提示词

请生成一张横版界面示意图，表现“多智能体群聊协作”场景。画面是一个正式的产品界面或操作台，不是聊天气泡堆砌图。

画面内容建议包含：
- 左侧智能体列表或成员头像区，标出最多 6 个文化 Agent
- 中间主聊天区，显示用户提问和多个 Agent 分工回答
- 右侧信息区，可展示知识范围、检索结果、可信度或引用来源
- 聊天内容体现“围绕同一非遗主题，多 Agent 分工回答”的特点

表现重点：
- 每个 Agent 有不同头像和名称
- 对话流式感强，像实时输出
- 突出“文化问答”“证据链”“知识增强”“流式回复”这些关键词

视觉要求：
- 界面清晰，层次分明
- 适合放入比赛说明书
- 颜色延续深蓝、青绿、少量紫色的科技文化风格

负面约束：
- 不要过度卡通化
- 不要无意义长文本
- 不要把输入框、按钮做得太小
- 不要让聊天内容遮挡界面结构
