# 图01 项目总览主视觉提示词

## 用途

用于生成说明书中的第一张核心插图，建议放在“作品简介”小节末尾。画面应当像一张正式比赛说明书主视觉，不要做成普通海报。

## 生成提示词

请生成一张横版 16:9 的比赛项目主视觉图，主题是“同源（OmniSource）”，表现一个面向福建非遗的数字生命共创平台。整体风格正式、清晰、现代，适合放入作品设计说明书。

画面主体包含：
- 福建地图意象或九地市文化分布意象
- 多个文化 Agent 的数字化形象或头像矩阵
- 真实产品能力暗示：Agent 聊天室、Agent 库、知识摘要、证据链、综合置信度、关系图谱、生图、3D 展示
- 适量科技感线条、界面卡片、数据流和文化纹样

构图要求：
- 左侧预留标题说明区，右侧作为主视觉区，或上下分区均可
- 画面要有层次，但不能太满
- 色彩以深蓝、青绿、少量金色或紫色点缀为主
- 风格偏比赛汇报、技术展示、文化传播，不要商业广告感

文字要求：
- 尽量避免大段小字
- 若出现文字，只保留“OmniSource / 非遗数字生命共创平台 / Agent 对话 / 证据链 / AIGC 共创”等少量清晰标题

负面约束：
- 不要杂乱
- 不要低清晰度
- 不要卡通幼稚风
- 不要商业宣传海报感
- 不要堆满无关元素
