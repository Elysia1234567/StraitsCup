# 图08 运行闭环示意图提示词

## 用途

用于“测试与运行效果”章节，展示项目从输入到展示的完整闭环。

## 生成提示词

请生成一张横版流程示意图，主题是“同源（OmniSource）运行闭环”。画面要像正式系统流程图，结构清晰，不要像宣传海报。

流程建议包含：
- 用户在 Web 或 UniApp 选择历史聊天室或新建聊天室
- 添加或删除房间 Agent
- 用户输入问题或上传图片
- 后端进行 Agent 编排
- 本地 JSONL RAG 检索
- 可选 Milvus 向量增强
- WebSocket 流式回复
- `/api/chat-rooms/{roomId}/insight` 生成知识摘要、关键证据、综合置信度和溯源信息
- AIGC 图像生成并写入聊天室
- OSS 归档
- 前端实时展示

表现方式建议：
- 从左到右的线性流程
- 每个节点标注简短功能说明
- 可在底部补一条“页面呈现”分支，显示首页、Agent 对话页、Agent 库弹窗、图谱页和 3D 页

视觉要求：
- 专业、简洁、适合答辩
- 字体清晰，模块边界明显
- 不要过多装饰性元素

负面约束：
- 不要画成复杂工程拓扑图
- 不要出现模糊英文长句
- 不要用过强霓虹光效
