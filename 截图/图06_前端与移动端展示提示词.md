# 图06 前端与移动端展示提示词

## 用途

用于展示 Web 端与 UniApp 移动端双端能力，建议放在“技术架构与技术路线”或“实用点”章节。

## 生成提示词

请生成一张横版双端展示图，左侧是 Web 端，右侧是移动端，表现同一个项目在两个端上的落地效果。画面应正式、整洁、适合比赛说明书。

Web 端建议包含：
- 福建地图入口
- 非遗画廊
- Agent 聊天室
- 关系图谱
- 3D / WebGL 展示区

移动端建议包含：
- 首页
- 聊天页
- 智能体页
- 图鉴页
- 接口实验室页

视觉要求：
- 两端风格一致
- 设备边框清晰
- 重点突出“同源（OmniSource）”的双端原型能力

负面约束：
- 不要做成手机广告图
- 不要把界面缩得看不清
- 不要出现杂乱背景
