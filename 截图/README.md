# 截图与提示词索引

本目录存放当前项目截图、校徽/Logo 资源，以及说明书配图和作品简介图的生成提示词。截图内容已按最新前端代码更新，Agent 页面以真实聊天室界面为准，不再使用旧版“多智能体群聊示意”界面。

## 当前截图素材

- `网页端Home.jpg`：Web 首页，福建地图入口与九地市非遗画廊。
- `网页端Agent对话.jpg`：Web Agent 真实聊天室，包含站点导航、成员列表、可增删 Agent、流式对话、历史聊天室入口、知识摘要、关键证据、综合置信度和溯源信息。
- `网页端Agent智能体库.jpg`：添加 Agent 弹窗，支持从 Agent 库选择并加入当前聊天室。
- `网页端Graph.jpg`：非遗关系图谱页面。
- `网页端3D模型.jpg`：WebGL / 3D 展示页面。
- `手机端首页.png`：UniApp 首页。
- `手机端Agent对话页.png`：UniApp Agent 对话页。
- `手机端智能体页.png`：UniApp 智能体页。
- `手机端图鉴页.png`：UniApp 图鉴页。
- `福州大学校徽.png`：说明书与简介图署名素材。
- `logo.svg`：项目图标素材，仅保留图标时优先使用该文件。
- `简介图宣传借鉴图片.png`：简介图版式参考素材。

## 使用建议

- 作品简介图优先参考 `AI生成作品简介图提示词.md`，并使用最新的 `网页端Agent对话.jpg`、`网页端Agent智能体库.jpg`、首页、图谱、3D 和移动端截图。
- 说明书中的图 03 应使用真实 Agent 对话界面截图或按该截图生成的产品界面图，标题建议写为“网页端 Agent 对话界面”。
- 图 04 重点说明知识摘要、关键证据、综合置信度和溯源信息的后端来源：`/api/chat-rooms/{roomId}/insight` 会综合消息元数据、RAG 检索、联网搜索与对话兜底生成。
- 当前数据库无需新增表即可支撑已实现功能；如后续要长期保存每次洞察快照，可新增 `chat_room_insight_snapshot`。
