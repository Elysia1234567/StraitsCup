## 设计目的

传统非遗数字化多停留在图文展示和资料检索，用户只能被动观看。《同源》把福建地域非遗转化为具有性格、知识边界和创作能力的“数字生命”，让用户从观看者变成对话者、共创者和传播者。

## 技术路线

前端采用 **Vue 3 + Vite + Tailwind CSS + ECharts + Three/OGL**，构建福建地图、非遗画廊、多智能体聊天室和 AI 生图界面。后端采用 **Spring Boot 3 + Spring AI + MyBatis + MySQL + Redis + Milvus + WebSocket + OSS**，形成“知识检索 → Agent 编排 → 流式生成 → 图像归档 → 实时呈现”的闭环。

## 目前实现功能

- **地域非遗入口**：地图联动画廊，展示福建九地市代表性非遗内容。
- **多智能体对话**：用户可选择文化 Agent 创建聊天室，支持历史会话、成员管理和流式回答。
- **RAG 知识增强**：基于非遗知识库进行向量检索，回答优先参考资料，减少文化内容幻觉。
- **AIGC 图像共创**：指定 Agent 按自身文化原型生成图像，结果可进入聊天室并归档。

## 创新性与实用性特点

1. **非遗器灵化表达**：为非遗项目设计名称、头像、性格、语言风格和知识范围，增强文化亲近感。
2. **多智能体协作叙事**：根据用户问题选择相关 Agent 发言，支持全员回答和接力讨论。
3. **RAG + Prompt 约束**：将知识片段、人设和输出规则统一组装，兼顾沉浸感与事实可靠性。
4. **生成式文化共创**：用户不仅能听讲解，还能围绕非遗原型生成视觉作品。
5. **可落地工程原型**：已具备前端页面、后端接口、数据库持久化、实时通信和图像存储能力。

## 海报推荐主文案

《同源》不是简单的非遗展示页，而是一套让非遗“会说话、会协作、会生成、可沉淀”的数字生命共创平台。
